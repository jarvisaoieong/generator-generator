# generator-docpad

> Yeoman generator for gruntfile using coffee-script

## Getting Started

```sh
$ npm install -g kinua/generator-gruntfile
$ mkdir /path/to/your/project
$ cd $_
$ yo gruntfile
$ grunt watch
```

Then happy coding with coffee-script.
